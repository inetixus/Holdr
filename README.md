# Holdr

Holdr is a React Native / Expo mobile MVP for tracking receipts, coupons, warranties, and their deadlines.

## Run it

1. Install Node.js LTS and Expo Go on an iPhone or Android phone.
2. In this folder, run `npm install` then `npm start`.
3. Scan the QR code with Expo Go.

The first version includes the urgent feed, adding a manual item, simulated photo/email intake, one-tap deadline correction, and marking an item used.
